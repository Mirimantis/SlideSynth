Overview

Glissandograph is a canvas-based music composition and performance tool where you draw Bezier curves on a chromatic staff to create melodies. Each curve maps pitch over time and is played back as a continuous, sliding tone — enabling expressive glissandi, portamenti, and microtonal passages that traditional step-sequencers can't easily produce. Beyond drawing, the canvas doubles as a performance surface: with Scroll Canvas on during Playback, the canvas slides past a stationary rail so you can play — and optionally record — live gestures with the mouse.

The workspace has three columns: a track panel on the left, the canvas in the centre, and a properties panel on the right. The right-hand panel is split into Tool Properties (variable settings for the active tool) on top and Object Properties (context for the selected track or point) below. The left panel is divided into Transport (BPM, time signature, metronome, loop, scroll mode), Tools (tool-mode buttons), and Tracks. A toolbar across the top shows the composition name (editable) alongside the composition length (auto-derived from the rightmost point, shown as M:SS), followed by scale controls and file operations. A ruler bar along the top of the canvas has two stacked strips: a seconds ruler on top that adapts its tick interval to zoom (sub-second decimals when zoomed in, whole seconds or larger when zoomed out) and a beat ruler below showing measure numbers and beat/subdivision ticks. Both strips are part of the playhead scrub zone.

Perform & Record

These features are under active development — behaviour may change.

The canvas doubles as a performance surface. Set the Scroll switch to Canvas and press Play: the music slides right-to-left past a stationary rail at the centre of the canvas. The planchette is the movable dot that rides the rail, following the mouse's Y to show the pitch currently being played. Arming Record captures your mouse gestures as editable Bezier curves on the selected track.

Coming later: a harmony/chord mode will add additional planchettes to the rail, each sounding a scale-appropriate interval above the primary.

Playing

  • Set the Scroll switch (transport panel) to Canvas and press Space or Play — the canvas scrolls past the rail at the composition's BPM.
  • Mouse Y moves the planchette along the rail, driving pitch (snapped to scale/semitone if snap is on).
  • Hold LMB while Playback is running to sound the selected track's tone at the planchette's pitch — release to stop. XY tools (Draw, Select, Delete, Slice) gray out while Perform owns LMB.
  • Play and Record both start at the rail's current beat — wherever you've panned to — not from beat 0.
  • With Loop on, playback uses the loop markers (dragged in the ruler) for its range.
  • With the Scroll switch on Planchette, Playback stays classic — the playhead moves across a fixed canvas and LMB runs the active XY tool as usual.

Recording

  • R or the red Record button arms recording. From Idle it runs a 3-second countdown then starts Playback. If Playback is already running, it arms immediately with no countdown. Recording forces the Scroll switch to Canvas.
  • While recording, each LMB press/release captures a curve onto the selected track. The curve is simplified and auto-smoothed on release, so the result is fully editable afterward.
  • Escape during the countdown or recording stops and returns to Idle.
  • Recording continues past the composition end so you can extend phrases or capture silence; an AFK timeout stops it after about 2 minutes of inactivity. The timeout is suppressed while Loop is on, while the playhead hasn't yet reached the rightmost control point (there's still existing content to record over), and while MIDI keys are being played.
  • Looping back (loop-wrap) during recording commits the current gesture as a curve and flashes the planchette — the next pass starts a fresh curve.

Heads-up display

  • Pitch readout (top-right) shows the planchette's current pitch as a note name plus cents offset (e.g. C#4 +12¢). Out-of-tune cents appear only when the raw cursor pitch differs from the snapped pitch by more than a cent.
  • Record button in the transport row pulses red when armed and glows solid red while recording.

Drawing Curves

Select the Draw tool (D) and click on the canvas to place control points. Each click adds a point to the current curve. Dragging after clicking a point pulls out Bezier handles to shape the curve between points.

Drawing details
  • Click on empty space to start a new curve.
  • Click on an existing point to reset its handles, then drag to reshape.
  • Press Escape or Enter to finish drawing the current curve.
  • While in Draw mode, hold Ctrl to temporarily switch to Select mode.
  • Bezier Auto-Smoothing (Tool Properties) — when on, a single click to place a point (second point onward) gives it horizontal handles sized at a fraction of the neighbouring segment length (controlled by the Handle length slider right below the checkbox, default 0.25). The first point of a new curve is unaffected. If you drag after clicking, the drag defines the handles as usual. Shift+S re-applies the same smoothing to the selected curve(s) using the current slider value; Alt+S clears handles for sharp corners.

Selecting & Editing

Switch to the Select tool (V) to manipulate existing curves and points.

Selection details
  • Click a point to select it — drag to move it.
  • Click a curve segment to select the whole curve.
  • Shift+click to add/remove curves from a multi-selection.
  • When a single curve is selected, drag its handles (handleIn / handleOut) to adjust curvature.
  • Hold Shift while dragging to constrain movement to the horizontal or vertical axis.
  • Right-click anywhere on the canvas to open an action menu with Smooth Curve, Sharpen Curve, and Join; items dim when no curves are selected.

Slice (Split Curve)

Select the Slice tool (C) and click on a curve segment to split it into two independent curves at the click location. The visual shape is preserved exactly — the two halves look identical to the original.

Slice details
  • A red preview dot shows where the cut will happen as you hover over a curve.
  • Click on a curve segment to split it into two curves at that point. Bezier handles are recalculated so both halves maintain the original shape.
  • Click near an interior control point to split exactly at that point, duplicating it as the end/start of each half.
  • When snap is enabled, the cut position snaps horizontally to the beat grid.
  • Volume is interpolated at the split point.
  • Use Ctrl+Z to undo a split.

Join Curves

Select two or more curves and press Ctrl+J to merge them into a single curve. Curves are joined in time order (left to right).

Join details
  • If adjacent endpoints are close together (within a few pixels), they merge into one point.
  • If endpoints are far apart, a new straight segment bridges the gap.
  • Curves that overlap in time (start point before the previous curve's end) are skipped.
  • Handles at merged points are preserved from each side, maintaining curve shape.
  • Also available via the right-click action menu on the canvas.
  • Use Ctrl+Z to undo a join.

Transform Box

Select multiple curves (via Shift+click) or click a single curve to activate a transform box with 8 resize handles.

Transform details
  • Drag corners or edges to resize the selection.
  • Drag the interior to translate all selected curves.
  • Use the octave shift arrows (▲ ▼) to shift pitch by ±12 semitones.
  • Hold Alt when you start dragging to duplicate the selection before transforming.

Playback & Transport

Use the transport buttons or press Space to toggle play/pause.

Playback details
  • Play / Pause / Stop — toolbar buttons or Space to toggle.
  • BPM — adjustable from 20 to 300.
  • Time signature — dropdown with 2/4, 3/4, 4/4, 5/4, 7/4, 6/8, 9/8, 12/8. Measure lines on the ruler and staff update immediately; BPM continues to mean quarter notes per minute regardless of denominator. Persisted with the composition.
  • Metronome — checkbox + volume slider in Transport. When enabled, audible clicks play during Playback (and Record) with three tiers: a bright high click on beat 1 of each measure (downbeat), a medium click on compound-meter secondary accents (beat 4 of 6/8, etc.), and a soft low click on every other tick. The ruler playhead / rail flashes a coloured ring on each tick so the beat is visible as well as audible. /4 meters tick on the beat; /8 meters tick on eighth-note subdivisions. Metronome clicks are not captured while recording (Record stores cursor Y, not audio output). Enabled state and volume persist via localStorage.
  • Length — derived automatically from the rightmost point in any curve and shown as M:SS next to the composition name. Playback stops at that point (or loops back to the start if Loop is on). With no points, playback is a no-op.
  • Canvas extent — the grid, staff, and rulers render everywhere you pan or zoom, with a generous buffer of open space past the last point so you can keep drawing outward. A memory-safe cap limits total pan range.
  • Loop (L) — when enabled, two draggable loop markers (green in, red out) appear in the ruler. Drag them to set the loop range; they snap to the beat grid and to existing curve control points. The range is saved with the composition. Markers hide when Loop is off; playback then runs through to composition end.
  • Scroll switch — chooses what scrolls during Playback. Set to Canvas (default) to slide the canvas right-to-left past a stationary centered rail, giving a performance feel. Set to Planchette for a classic static canvas where the playhead moves across a fixed view — useful for A/B editing. Preference persists via localStorage. Record forces it to Canvas while armed.
  • Pitch HUD — shows the pitch under the cursor (note name + cents) in the top-right of the canvas while the cursor has a meaningful Y. Toggle off to hide. Preference persists via localStorage.
  • Record (R) — arms recording. From Idle, a 3-second countdown runs then Playback begins. If Playback is already running (classic or Perform), the countdown is skipped and recording arms immediately at the current beat. While armed, holding LMB in the canvas plays the selected track's tone AND captures it as a curve; release commits the curve. Recording forces the Scroll switch to Canvas and hijacks LMB for Perform — the XY tool buttons (Draw, Select, Delete, Slice) gray out while live. Escape cancels.
  • Perform during Playback — when Playback is running with the Scroll switch on Canvas (including during Record), LMB plays the selected track's tone at the cursor's Y pitch. XY tools are disabled for the duration. Release LMB to stop the tone. With the switch on Planchette, Playback stays classic — LMB runs the active XY tool as usual.
  • Ruler — two stacked strips at the top of the canvas: a seconds ruler (adapts tick spacing to zoom; labels decimal seconds when zoomed in and M:SS for larger steps) and a beat / measure ruler with subdivision ticks. The seconds ruler also reflects BPM changes.
  • Scrubbing — click and drag anywhere in the ruler bar (either strip) to position the playhead. Scrubbing is allowed into empty grid past the last point.

Tracks & Tones

The track panel on the left lets you manage multiple tracks, each with its own tone, colour, and curves.

Track details
  • Click + Track to add a new track.
  • Click + Tone to open the tone builder and design a custom synth voice.
  • Each track has M (mute), S (solo), and T (edit tone) controls.
  • Tone builder supports oscillator types (sine, square, sawtooth, triangle), detune, gain layers, and optional distortion.
  • Four built-in presets: Pure Sine, Bright Square, Warm Pad, Buzzy Saw.

Live Audio Preview

Hold Space to hear a live preview of the note at the cursor position. This lets you audition pitches before placing points, making melodic exploration much faster.

Preview details
  • Draw preview — in Draw mode, hold Space with the cursor over the canvas. The Tool Properties panel offers two modes:
    • Tone only (default) — hear the current track's tone at the cursor's pitch. Move the cursor up and down to modulate pitch in real time.
    • Composition + tone — hear all existing curves evaluated at the cursor's X position together with the cursor tone at the cursor's Y pitch. This previews what the composition would sound like if a point were placed at the cursor. Move left/right to scrub through time and up/down to audition the new pitch against the rest of the composition.
  • Scrub preview — while click-dragging in the ruler to scrub, hold Space to hear all active curves at the current playhead position. Drag left/right to hear how the composition sounds at each point in time.
  • Pitch respects snap and scale settings — if snap is on with a scale selected, you'll hear scale-quantised notes.
  • Release Space to stop the preview.
  • When the cursor is not over the canvas or you're not in Draw mode, Space works as the normal play/pause toggle.

Live MIDI Input

Connect a MIDI keyboard or controller and play notes straight into a track's tone — and optionally record what you play as curves. Distinct from File > Import MIDI, which reads an existing .mid file into new curves.

MIDI input details
  • Pick a device from the MIDI Input dropdown in the Transport panel. On first selection the browser prompts for MIDI access.
  • Note On sounds a track's tone at the note's pitch; Note Off stops it. Held notes play in parallel (polyphonic), so you can sound chords.
  • Audio target: when no track is MIDI-armed (see below), notes sound through the selected track's tone. When a track is MIDI-armed, the armed track's tone takes over so what you hear matches what would be recorded.
  • The dropdown updates automatically when devices are plugged or unplugged.
  • Pitch-bend and CC messages are ignored for now.
  • Requires a browser with Web MIDI support (Chromium-based browsers). The dropdown is disabled when the API is unavailable.
Recording MIDI input to curves

Each track row has an I button (alongside Mute / Solo / Edit Tone) that arms the track for MIDI input recording. While playback is running, MIDI notes captured into the armed track land as curves on release — separate per note, so chords record as multiple parallel curves on the same track.

  • Mutually exclusive — clicking I on a different track disarms the previous one. Click the same track's I again to disarm.
  • The I button shows the current state: hollow when disarmed, solid grey when armed-but-quiet, pulsing red while one or more MIDI notes are actively recording into that track.
  • Distinct from the global Record button (which arms LMB-held perform recording). Both can be active in parallel — your mouse perform records to the selected track while MIDI records to the armed track, in independent voices.
  • Pitch is not snapped — MIDI notes record at their exact MIDI pitch (no scale or grid quantization).
  • Recording requires playback to be running. Arming a track and pressing Play from idle gives you the same countdown affordance as the LMB record-arm flow.
  • Stop boundaries (Stop / Pause / Escape / loop wrap / disarming the track) finalize any in-flight notes cleanly so you don't lose mid-note captures.

Snap & Grid

Toggle snap on/off with the Snap switch in the Transport panel or press S.

Snap details
  • When snap is on, points snap to the time grid and the nearest semitone (or nearest scale note if a scale is active). Transitions between snap lines are instantaneous unless Magnetic mode (below) is also on.
  • Adaptive snap — the time-axis snap granularity adjusts with zoom: 1/16-beat at high zoom, eighth-note at medium zoom, and whole beats when zoomed out. The ruler ticks match the current snap resolution.
  • Bezier handles are never snapped, allowing smooth curve shaping regardless of snap setting.
  • Hold Shift while dragging to constrain movement to one axis.
  • Magnetic — a spring-mass physics alternative for pitch snapping during performance. The planchette has inertia, is pulled toward the cursor by an elastic force, and is attracted to the nearest snap line with proximity-based strength. Only active during Compose Performance (LMB held). Requires Snap on to have any effect. Three tuning sliders:
    • Magnetic (0–1, default 0.75) — snap attractor strength. At 0 the planchette is a smooth cursor-follower with no snap detents; higher values make snap lines pull the pitch in increasingly hard.
    • Spring (1–50, default 30) — cursor-to-pitch coupling stiffness. Lower values feel rubbery and let the planchette linger between snaps; higher values track the cursor tightly.
    • Damping (0.25–15, default 3) — how quickly pitch oscillations near a snap line fade. Low values sustain tremolo wobbles for longer; high values settle quickly. Too low combined with high Magnetic can swing wildly.
    Combine strong Magnetic with low Damping and a moderate Spring to get tremolo around held notes; combine low Magnetic with high Spring for free-sliding microtonal performance with gentle detents.

All snap-section settings (Snap toggle, Magnetic on/off, the three magnetic sliders, plus the Key & Scale picks) are saved with the composition file and restored on load, so a project's bespoke snap configuration round-trips cleanly.

Snap presets

The Preset dropdown at the top of the Snap section loads a named bundle of snap settings in one click. Built-in presets cover the common workflows:

  • Free Draw — snap off; the cursor follows freely (good for sketching).
  • Chromatic 1/16 — snap on, magnetic off; the classic discrete grid.
  • Magnetic On-Pitch — tight magnetic pull (strength 0.85, spring 35, damping 4); great for pitch-locked performances.
  • Magnetic + Diatonic — looser magnetic feel (strength 0.7, spring 22, damping 2.5); pair with a Key/Scale for diatonic snap with vibrato headroom.

Click Save to capture the current settings as a named user preset (stored locally in your browser). Del removes the active user preset. Presets only override snap/magnetic fields — they leave your Key/Scale pick untouched. Loading a preset is not recorded in undo history (it's a workflow toggle, not a composition edit).

Snap guides

Snap guides are user-placed lines that act as additional snap targets, on top of the regular grid / scale. Use the + X and + Y buttons in the Snap section to drop a vertical (beat) or horizontal (pitch) guide at the centre of the viewport.

  • Switch to the Select tool, then click on a guide line to select it. The Object Properties panel then shows a label field (which renders inline on the guide) plus a Delete button. (Other tools like Draw fall through to their normal click action so guides don't get in the way of authoring.)
  • Drag a selected guide to move it (snaps to grid / scale, with other guides as additional targets).
  • Press Delete with a guide selected to remove it.
  • The Guides toggle hides all guides and disables their snap pull at once. Hidden guides aren't deleted; flip the toggle back on to bring them back.
  • The Lock toggle (next to Guides) prevents existing guides from being selected, dragged, or deleted on the canvas. Snap pull still works while locked. Use it to protect a guide layout from accidental edits while you're authoring curves.
  • During Compose Performance / Record, the planchette pitch snaps to Y-guides as additional targets (in addition to the regular grid / scale / magnetic system). X-guides only affect placement-style snapping (Draw / Select), not perform-time progression.
  • Guides are saved with the composition. Their visibility and lock state, however, are per-browser preferences (so you can hide or unlock temporarily without changing the file).
  • Harmonic Prism's projection echoes still take exclusive snap ownership when active — guides defer so projection mode stays predictable.

Key & Scale

Use the Key and Scale dropdowns in the toolbar to set a musical key. When a key is active, the staff highlights in-scale notes and snap quantises to scale degrees.

Scale details
  • 30 scales across 5 groups: Western modes, pentatonic/blues, other Western, world scales, and microtonal.
  • Microtonal scales (e.g. Maqam Rast, Gamelan Slendro) use fractional semitone intervals.
  • In-scale accidentals (sharps/flats that are part of the current scale) are visually prominent on the staff.
  • Chromatic (default) — no key. All 12 semitones render as background staff lines and Y-snap pulls to the nearest semitone.
  • None — clean canvas. Hides every default semitone line and turns off the chromatic Y-snap, leaving the cursor free in pitch. Other snap targets still work: user-placed pitch guides, Harmonic Prism projection echoes, and curve/point snapping all keep pulling. Useful when you've set up custom pitch guides and want only those to be visible and active.

Harmonic Prism

The Harmonic Prism turns single-pitch curves into chord gestures. Two modes share one chord-spec engine:

  • Draw Mode (H) — clicking with the Draw tool places N grouped sibling curves at chord-related Y offsets. The cursor shows a multi-color preview of where each voice will land.
  • Projection Mode (Ctrl+H) — visual "echoes" of a selected curve are projected up and down the canvas at harmonic intervals. Echoes are silent but snappable — you can draw or perform new curves along them. While projection is active, snap locks to echo pitches only.
Using Draw Mode
  1. Pick the Draw tool. Toggle Draw on in the Harmonic Prism panel (or press H).
  2. Hover the canvas. The cursor shows a primary planchette (rainbow-filled disc) plus N−1 harmony planchettes (solid rainbow stops) at the chord-spec Y offsets.
  3. Click + drag once. N sibling curves appear with parallel handles, sharing a single group id.
  4. Click again to extend the cluster — every sibling gets a new point at the cursor X with its harmony offset.
  5. Selecting one sibling selects the whole group; the Transform Box wraps the cluster and resize / translate / octave-shift affect all voices.
  6. Editing one sibling's points (in the Select tool) does not cascade — grouping is for placement, deletion, and transforms, not point-level edits. Use Projection mode if you want locked parallelism.
Performing & recording chord clusters

With Draw mode on, holding LMB on the canvas sounds all N voices simultaneously — primary at the cursor pitch, harmonies at the chord-spec offsets above (or around) it. The rail shows every voice as a planchette so you can see the cluster's shape while you play.

  • Move the cursor while LMB is held — every voice transposes together; chord shape stays constant.
  • Harmonies that fall outside the staff range are silenced and hidden from the rail until they come back into range.
  • If you change tuning (12-TET ↔ Just Intonation) or chord quality while still holding LMB, the active harmonies retune live to the new spec.
  • Voice count is locked at LMB-down — switching to a higher voice count mid-hold doesn't add new voices to the active gesture. Release and press again to engage the new count.
  • Arm Record (R), then hold LMB and play — every voice records to its own curve buffer. On release, the N captured curves commit as a chord cluster (same group id, one undo step) onto the active track.
Using Projection Mode
  1. Draw a single curve (the "source").
  2. Select it (click with the Select tool).
  3. Press Ctrl+H. Dashed lavender echoes appear. The staff stays visible so you can read absolute pitch.
  4. Tune the chord spec in the Harmonic Prism panel (left sidebar).
  5. Draw a new curve — the cursor snaps exclusively to echo pitches for the duration of projection.
  6. Press Ctrl+H or Esc to clear projection. Normal snap resumes.
Group / Ungroup

Chord-cluster siblings drawn in Draw Mode share a group id. Group is also a first-class freehand action you can apply to any selection of curves on a track:

  • Ctrl+G — group ≥2 selected curves so they move, delete, copy, paste, and transform as a unit.
  • Ctrl+Shift+G — dissolve the group on every selected curve (and its siblings); the curves become independent again.
  • Both actions are also in the right-click context menu.
  • Copy / paste / duplicate / continue / Alt+drag preserve group identity within the new copy with a fresh id, so you can build up multiple independent clusters from one original.
  • Splitting a grouped curve with the Slice tool keeps both halves in the same group. Joining two curves from different groups is refused (a toast explains why).
Chord spec controls
  • Stacking — the interval used to stack voices. Tertian (3rds) for conventional harmony, Quartal (4ths) for modal/jazz, Quintal (5ths) for open sound, Secondal (2nds) for clusters.
  • Quality — major / minor / dominant / diminished / augmented / sus2 / sus4. Fully applies to tertian; limited options for other stackings.
  • Voices — 2 (dyad) through 5 (pentad). More voices extend the chord: triad → 7th → 9th.
  • Tuning — 12-TET (grid-aligned) or Just Intonation (pure acoustic ratios).
  • Direction — voices sit above the base, below, or around it.
  • Octaves ± — how many octaves above and below the source to echo (0–3).
A brief acoustics primer

Harmonic series

Every pitched sound is a stack of integer-multiple frequencies (1×, 2×, 3×, …). Intervals that sound consonant are simple ratios between these multiples — the octave is 2:1, the perfect fifth is 3:2, the major third is 5:4.

12-TET vs Just Intonation

12-TET (equal temperament): the octave is split into 12 logarithmically-equal semitones. Every key sounds equally acceptable — and equally compromised. The 12-TET major third is 400¢.

Just Intonation: use the acoustic ratio directly. The JI major third (5/4) is 386.31¢ — about 14¢ flatter than 12-TET, but it "locks" on sustained tones without the beating you hear in equal temperament. SlideSynth's JI dominant 7th uses the harmonic seventh (7/4 ≈ 969¢), noticeably flatter than the 12-TET m7 (1000¢).

Why microtonal matters

SlideSynth pitches are continuous floats, not quantised to semitones. A chord built at MIDI 60.4 is just as valid as one at 60 — the harmonic maths is invariant to the base pitch.

Stacking styles

Tertian is conventional Western harmony (stacked 3rds). Quartal stacks 4ths (think modal jazz, McCoy Tyner). Quintal stacks 5ths (open, power-chord-like). Secondal stacks 2nds (cluster chords).

Snapping to echoes
  • While Projection is active, snap locks exclusively to echo pitches. Staff lines and scale snapping are suspended so you can land precisely on a JI interval between staff lines.
  • X still snaps to the beat grid as usual.
  • The staff stays visible as a reference — use it to read absolute pitch, but trust the snap to your ear.
  • Turn projection off (H or Esc) and normal staff/scale snapping returns.

Zoom & Pan

Navigation controls
  • Zoom Y (pitch) — vertical slider at bottom-right of canvas, or scroll wheel.
  • Zoom X (time) — horizontal slider at bottom-right of canvas, or Ctrl+scroll wheel.
  • Pan — Alt+click drag or middle-click drag.
  • Zoom is centered on the cursor position.

Copy, Paste & Duplicate

Clipboard operations
  • Ctrl+C — copy selected curves.
  • Ctrl+X — cut selected curves.
  • Ctrl+V — paste at the current playhead position.
  • Ctrl+D — duplicate selected curves in place.
  • Ctrl+Shift+D — continue drawing (append a copy to the end of the selection).

Undo & Redo

History controls
  • Ctrl+Z — undo (up to 50 steps).
  • Ctrl+Shift+Z or Ctrl+Y — redo.
  • Toolbar Undo / Redo buttons mirror the shortcuts.

File Menu

File operations
  • Composition name — the editable text field at the top-left of the toolbar sets the composition name, which is used as the filename when saving.
  • Click the File button next to the composition name to open the file menu.
  • File > Save Composition — downloads the composition as a JSON file named after the composition.
  • File > Load Composition — opens a JSON composition file (the name field updates to match).
  • File > Import MIDI — imports a MIDI file, converting notes to Bezier curves with pitch bends.
  • File > Export WAV — renders the composition to a 16-bit stereo WAV file (44100 Hz).
