Overview

SlideSynth is a canvas-based music composition tool where you draw Bezier curves on a chromatic staff to create melodies. Each curve maps pitch over time and is played back as a continuous, sliding tone — enabling expressive glissandi, portamenti, and microtonal passages that traditional step-sequencers can't easily produce.

The workspace has three columns: a track panel on the left, the canvas in the centre, and a properties panel on the right. The right-hand panel is split into Tool Properties (variable settings for the active tool) on top and Object Properties (context for the selected track or point) below. A toolbar across the top shows the composition name (editable) alongside the composition length (auto-derived from the rightmost point, shown as M:SS), followed by tool selection, zoom, snap, scale controls, and file operations. A ruler bar along the top of the canvas has two stacked strips: a seconds ruler on top that adapts its tick interval to zoom (sub-second decimals when zoomed in, whole seconds or larger when zoomed out) and a beat ruler below showing measure numbers and beat/subdivision ticks. Both strips are part of the playhead scrub zone.

Glissandograph Mode

This mode is under active development — behaviour may change.

A performance mode where you play the canvas like an instrument. The music scrolls right-to-left past a stationary planchette rail in the middle of the canvas. The planchette is the movable indicator that rides the rail, following the mouse's Y position to show the pitch currently being played. Switch modes via the Compose / Glissandograph pill buttons in the toolbar. Toolbar, tracks, snap, key, and scale all behave the same in both modes.

Coming later: a harmony/chord mode will add additional planchettes to the rail, each sounding a scale-appropriate interval above the primary.

Playing

  • Mouse Y moves the planchette along the rail, driving pitch (snapped to scale/semitone if snap is on).
  • Hold LMB to sound the selected track's tone at the planchette's pitch — release to stop. While idle, the Draw Preview mode (Tool Properties panel) decides whether "Composition + tone" also plays existing curves at that spot so you can hear the chord.
  • Space starts/stops scrolling play. The canvas slides right-to-left past the rail at the composition's BPM.
  • Play and Record both start at the rail's current beat — wherever you've panned to — not from beat 0.
  • With Loop on, playback uses the loop markers (dragged in the ruler, shared with Compose mode) for its range.
  • During play, holding LMB plays along but does not record (unless Record is armed).

Recording

  • R or the red Record button arms recording with a 3-second countdown, then starts scrolling play.
  • While recording, each LMB press/release captures a curve onto the selected track. The curve is simplified and auto-smoothed on release, so the result is fully editable back in Compose mode.
  • Escape during the countdown cancels.
  • Recording continues past the composition end so you can extend phrases or capture silence; an AFK timeout stops it after a minute of inactivity.

Panning and zooming

  • Middle-click drag pans. When idle, both X and Y pan freely; during play/record, only Y pans (X is owned by the scrolling playhead).
  • Beat 0 can sit at the rail — scroll or pan past the left edge to align it.
  • Mouse wheel zooms Y (pitch axis), including during scrolling play. Ctrl+wheel zooms X when idle; it's disabled during scrolling play so the canvas keeps tracking the audio.
  • Y auto-scroll — during play/record, if the planchette nears the top or bottom edge, the viewport pans in that direction so higher/lower notes come into view. Suspends briefly after an explicit Y-zoom so it doesn't fight your intent.

Heads-up display

  • Pitch readout (top-right) shows the planchette's current pitch as a note name plus cents offset (e.g. C#4 +12¢). Out-of-tune cents appear only when the raw cursor pitch differs from the snapped pitch by more than a cent.
  • Record button in the transport row pulses red when armed and glows solid red while recording.

Drawing Curves

Select the Draw tool (D) and click on the canvas to place control points. Each click adds a point to the current curve. Dragging after clicking a point pulls out Bezier handles to shape the curve between points.

Drawing details
  • Click on empty space to start a new curve.
  • Click on an existing point to reset its handles, then drag to reshape.
  • Press Escape or Enter to finish drawing the current curve.
  • While in Draw mode, hold Ctrl to temporarily switch to Select mode.
  • Bezier Auto-Smoothing (Tool Properties) — when on, a single click to place a point (second point onward) gives it horizontal handles sized at 50% of the neighbouring segment length, producing a smooth curve without needing to drag. The first point of a new curve is unaffected. If you drag after clicking, the drag defines the handles as usual.

Selecting & Editing

Switch to the Select tool (V) to manipulate existing curves and points.

Selection details
  • Click a point to select it — drag to move it.
  • Click a curve segment to select the whole curve.
  • Shift+click to add/remove curves from a multi-selection.
  • When a single curve is selected, drag its handles (handleIn / handleOut) to adjust curvature.
  • Hold Shift while dragging to constrain movement to the horizontal or vertical axis.

Slice (Split Curve)

Select the Slice tool (C) and click on a curve segment to split it into two independent curves at the click location. The visual shape is preserved exactly — the two halves look identical to the original.

Slice details
  • A red preview dot shows where the cut will happen as you hover over a curve.
  • Click on a curve segment to split it into two curves at that point. Bezier handles are recalculated so both halves maintain the original shape.
  • Click near an interior control point to split exactly at that point, duplicating it as the end/start of each half.
  • When snap is enabled, the cut position snaps horizontally to the beat grid.
  • Volume is interpolated at the split point.
  • Use Ctrl+Z to undo a split.

Join Curves

Select two or more curves and press Ctrl+J to merge them into a single curve. Curves are joined in time order (left to right).

Join details
  • If adjacent endpoints are close together (within a few pixels), they merge into one point.
  • If endpoints are far apart, a new straight segment bridges the gap.
  • Curves that overlap in time (start point before the previous curve's end) are skipped.
  • Handles at merged points are preserved from each side, maintaining curve shape.
  • Also available via the Join button in the toolbar.
  • Use Ctrl+Z to undo a join.

Transform Box

Select multiple curves (via Shift+click) or click a single curve to activate a transform box with 8 resize handles.

Transform details
  • Drag corners or edges to resize the selection.
  • Drag the interior to translate all selected curves.
  • Use the octave shift arrows (▲ ▼) to shift pitch by ±12 semitones.
  • Hold Alt when you start dragging to duplicate the selection before transforming.

Playback & Transport

Use the transport buttons or press Space to toggle play/pause.

Playback details
  • Play / Pause / Stop — toolbar buttons or Space to toggle.
  • BPM — adjustable from 20 to 300.
  • Length — derived automatically from the rightmost point in any curve and shown as M:SS next to the composition name. Playback stops at that point (or loops back to the start if Loop is on). With no points, playback is a no-op.
  • Canvas extent — the grid, staff, and rulers render everywhere you pan or zoom, with a generous buffer of open space past the last point so you can keep drawing outward. A memory-safe cap limits total pan range.
  • Loop (L) — when enabled, two draggable loop markers (green in, red out) appear in the ruler. Drag them to set the loop range; they snap to the beat grid and to existing curve control points. The range is saved with the composition. Markers hide when Loop is off; playback then runs through to composition end.
  • Scroll Canvas — when on (default), Playback scrolls the canvas right-to-left past a stationary centered playhead, matching the Glissandograph feel. Turn off for a classic static canvas where the playhead moves across a fixed view — useful for A/B editing. Preference persists via localStorage. The Glissandograph is always scrolling (Performance mode), so this toggle only affects Compose mode.
  • Ruler — two stacked strips at the top of the canvas: a seconds ruler (adapts tick spacing to zoom; labels decimal seconds when zoomed in and M:SS for larger steps) and a beat / measure ruler with subdivision ticks. The seconds ruler also reflects BPM changes.
  • Scrubbing — click and drag anywhere in the ruler bar (either strip) to position the playhead. Scrubbing is allowed into empty grid past the last point.

Tracks & Tones

The track panel on the left lets you manage multiple tracks, each with its own tone, colour, and curves.

Track details
  • Click + Track to add a new track.
  • Click + Tone to open the tone builder and design a custom synth voice.
  • Each track has M (mute), S (solo), and T (edit tone) controls.
  • Tone builder supports oscillator types (sine, square, sawtooth, triangle), detune, gain layers, and optional distortion.
  • Four built-in presets: Pure Sine, Bright Square, Warm Pad, Buzzy Saw.

Live Audio Preview

Hold Space to hear a live preview of the note at the cursor position. This lets you audition pitches before placing points, making melodic exploration much faster.

Preview details
  • Draw preview — in Draw mode, hold Space with the cursor over the canvas. The Tool Properties panel offers two modes:
    • Tone only (default) — hear the current track's tone at the cursor's pitch. Move the cursor up and down to modulate pitch in real time.
    • Composition + tone — hear all existing curves evaluated at the cursor's X position together with the cursor tone at the cursor's Y pitch. This previews what the composition would sound like if a point were placed at the cursor. Move left/right to scrub through time and up/down to audition the new pitch against the rest of the composition.
  • Scrub preview — while click-dragging in the ruler to scrub, hold Space to hear all active curves at the current playhead position. Drag left/right to hear how the composition sounds at each point in time.
  • Pitch respects snap and scale settings — if snap is on with a scale selected, you'll hear scale-quantised notes.
  • Release Space to stop the preview.
  • When the cursor is not over the canvas or you're not in Draw mode, Space works as the normal play/pause toggle.

Snap & Grid

Toggle snap on/off with the Snap button or press S.

Snap details
  • When snap is on, points snap to the time grid and the nearest semitone (or nearest scale note if a scale is active).
  • Adaptive snap — the time-axis snap granularity adjusts with zoom: 1/16-beat at high zoom, eighth-note at medium zoom, and whole beats when zoomed out. The ruler ticks match the current snap resolution.
  • Bezier handles are never snapped, allowing smooth curve shaping regardless of snap setting.
  • Hold Shift while dragging to constrain movement to one axis.

Key & Scale

Use the Key and Scale dropdowns in the toolbar to set a musical key. When a key is active, the staff highlights in-scale notes and snap quantises to scale degrees.

Scale details
  • 30 scales across 5 groups: Western modes, pentatonic/blues, other Western, world scales, and microtonal.
  • Microtonal scales (e.g. Maqam Rast, Gamelan Slendro) use fractional semitone intervals.
  • In-scale accidentals (sharps/flats that are part of the current scale) are visually prominent on the staff.
  • Set Key to None to return to chromatic mode.

Zoom & Pan

Navigation controls
  • Zoom Y (pitch) — vertical slider at bottom-right of canvas, or scroll wheel.
  • Zoom X (time) — horizontal slider at bottom-right of canvas, or Ctrl+scroll wheel.
  • Pan — Alt+click drag or middle-click drag.
  • Zoom is centered on the cursor position.

Copy, Paste & Duplicate

Clipboard operations
  • Ctrl+C — copy selected curves.
  • Ctrl+X — cut selected curves.
  • Ctrl+V — paste at the current playhead position.
  • Ctrl+D — duplicate selected curves in place.
  • Ctrl+Shift+D — continue drawing (append a copy to the end of the selection).

Undo & Redo

History controls
  • Ctrl+Z — undo (up to 50 steps).
  • Ctrl+Shift+Z or Ctrl+Y — redo.
  • Toolbar Undo / Redo buttons mirror the shortcuts.

File Menu

File operations
  • Composition name — the editable text field at the top-left of the toolbar sets the composition name, which is used as the filename when saving.
  • Click the File button next to the composition name to open the file menu.
  • File > Save Composition — downloads the composition as a JSON file named after the composition.
  • File > Load Composition — opens a JSON composition file (the name field updates to match).
  • File > Import MIDI — imports a MIDI file, converting notes to Bezier curves with pitch bends.
  • File > Export WAV — renders the composition to a 16-bit stereo WAV file (44100 Hz).
